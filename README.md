# ArduinoBinaryLedCounter
:zap: Just a simple 8 bit counter that counts till 255. Tested on Arduino Uno :zap:

### Technical:
* Arduino Uno with usb
* Breadboard / printboard
* Jump wires (11x)
* Pushbutton/switch
* 8 generic LEDS
* 1x 10k Ohm resistor
* 8x 1k Ohm resistor
